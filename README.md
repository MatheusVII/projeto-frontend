---

## ⚙️ Como rodar o projeto

Clone o repositório:

```bash
git clone https://github.com/MatheusVII/projeto-frontend.git
```

Entre na pasta:

```bash
cd frontend-projeto
```

Instale as dependências:

```bash
npm install
```

Rode o projeto:

```bash
npm run dev
```

---

## 📦 Build para produção

```bash
npm run build
```
